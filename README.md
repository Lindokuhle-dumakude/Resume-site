# Developer Resume Platform 💻🌐

A responsive, multi-page developer resume website 🧑🏽‍💻 built as a real-world learning project.
This project is designed to grow in phases 📈 as new skills are learned, starting with semantic HTML and modern CSS 🎨, and later expanding into JavaScript ⚙️ and full-stack features 🔧.

---

## 📌 Project Overview

The Developer Resume Platform is a personal portfolio-style website 🌍 that showcases:

- Personal profile and introduction 👋
- Skills and experience 🛠️
- Education and work history 🎓
- Contact information 📬

---

## 🎯 Project Goals

Practice semantic HTML and proper document structure 🏗️

- Build responsive layouts using modern CSS 📱
- Apply Flexbox and Grid appropriately 📐
- Create a scalable spacing system using CSS variables 🎚️
- Follow real-world frontend best practices 🌟
- Build a foundation that can later support JavaScript and backend features 🔌

---

## ✨ Features

- Semantic HTML structure 🌐
- Accessible navigation and content structure with keyboard support ♿
- Multi-page layout with separate home and contact pages 📄
- Proper use of lists, headings, and links for clarity and accessibility 🔗

- Responsive layout built with Flexbox and CSS Grid 📦🧩
- Mobile-first design approach with multiple responsive breakpoints 📱
- Structured and scalable typography using `rem` units 🔠
- Reusable spacing system implemented with CSS custom properties 📐

- Visible hover, focus, and active states for improved usability 🎯
- Keyboard-accessible navigation with clear focus indicators ⌨️
- Clean, maintainable, and well-commented CSS architecture 🧼

- Custom font integration using Google Fonts (Inter) ✍️
- Modern CSS reset for consistent styling across browsers 🔄
